# sorteioMesaDoFliper

sorteioMesaDoFliper é um script em Javascript que permite o usuário captar os donos de comentários específicos do chat do Youtube durante lives. Foi desenvolvido em específico para o [Canal GameFM](https://www.youtube.com/user/CanalGameFM) segundo as necessidades do mesmo, porém o script funciona para qualquer live no YouTube de acordo com layout de 2017.

## Como usar

Este script foi desenvolvido no console do Google Chrome, então tende a funcionar melhor nele.

 - Para utilizar basta abrir a janela de chat separada nas lives do
   Youtube;
   
 -  Após abrir a janela, pressione F12 para abrir as ferramentas de
   desenvolvedor do Chrome;
   
-   Copie o código fonte do arquivo sorteioMesaDoFliper*.js, cole no
   console e execute o script;
   
-   Após isso, siga as instruções do script que perguntará a quantidades
   de chaves a ser procuradas e quais são as mesmas;
   
-   Com as chaves configuradas, aguarde o tempo de 20 segundo que o
   script apresentará uma nova janela com usuários que comentaram a
   chave desejada.
